# Base UI kit package

This UI kit is individual and flexible. If it doesn't fit with your app then you can fork it yourself...

## Built With

- [Flutter](https://flutter.dev/)
- [Dart](https://dart.dev/)

### Add package to your project

```sh
dependencies:
  dev_ui_kit:
    git:
      url: https://github.com/danifdev/dev_ui_kit.git
      ref: ui_kit
```
